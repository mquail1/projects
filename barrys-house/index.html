Project Description

Barry's House is a Unity TwitchLib project : it connects the Unity build to the client's Twitch channel.

The purpose of the project is to allow the client's Twitch audience to influence the Unity environment through Twitch exclusive events, such as spending money to cheer bits, subscribing to the client's channel, etc.

The project build was featured live on the client's channel on April 1st, 2023.

Development spanned about 2-3 weeks.

3D Models

The bulk of my contributions were 3D models that served as set dressings for the various scenery.

The client's specifications were to keep the models low-poly.

Everything was modeled, textured and animated in Blender 3D software.
