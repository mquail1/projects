3D Project Proposal

3D Modeling, Unity shaders, Scripting

Project Description

My Senior Project proposal in Unity 3D.

The purpose of the project was to show basic proficiency in the Unity engine, along with the capability to produce primitive 3D objects, Unity shaders, and scripting.

The project proposal build was presented live in class in October 2022.

Development spanned about 4-5 days.

Shaders

My goal for the shaders was to evoke a futuristic, glowing outline.

The shaders used a transparent base and combine black and white textures with an alpha mask to achieve the look.

Finally, post-processing was used to render the HDR colors & emission to a soft glowing bloom.

3D Model

Finally, I decided to have a bit of fun and model a "Narrator AI" character.

The shader is a scrolling "holographic" texture using the same techniques as the glow shaders above, plus a texture plugin that scrolls.

The character has 3 animations done natively in Blender: idle, talking, and bowing.
