Game Design, 3D Modeling, Unity shaders, Scripting

Project Description

My favorite solo Unity project, combining live music and game design into one build.

The purpose of the project was to incorporate a live DJ show with TwitchLib and Unity integration to allow Twitch users to influence the show in real time.

The project build was premiered on January 7th, 2023 and again on February 12th, 2023 as an encore.

Development spanned about 2-3 weeks.

A Star is Born

I wanted to push myself squarely out of my comfort zone - by creating a fully functioning avatar for the show.

I modeled, rigged, textured, and animated everything myself, starting in Blender and then importing into Unity.
